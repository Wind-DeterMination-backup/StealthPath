# Detail

## 这一层是什么
`src/main/resources/` 是模组运行时资源层。

## 这一层具体干了什么
- 提供 `bundles/` 多语言资源。
- 运行时归档中还会携带根目录 `mod.json`，但当前仓库权威元数据文件已经放在根目录，而不是本目录。

## 与其他层级的关系
- 被 Java 代码通过 `Core.bundle` 和模组加载器读取。
- 最终会被复制进 `bin/main/`、`build/resources/` 和发布归档。

## 当前状态
- 历史自动文档仍把 `mod.json` 视为本目录文件，说明文档系统对资源布局的认识比真实仓库落后一步。
